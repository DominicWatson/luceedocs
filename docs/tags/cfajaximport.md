# cfAjaxImport


Controls the JavaScript files that are imported for use on pages that use Railox AJAX tags and features.

---
## Body
free

## Example
```
<cfAjaxImport
 [adapter="string"]
 [csssrc="string"]
 [params="struct"]
 [scriptsrc="string"]
 [tags="string"]
> 
 [</cfAjaxImport>]
```
