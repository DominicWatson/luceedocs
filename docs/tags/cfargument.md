# cfargument


Allows subtag data to be saved with the base tag. Applies only to custom tags.

prohibited

```
<cfargument
 [default="any"]
 [displayname="string"]
 [hint="string"]
 name="string"
 [passby="string"]
 [required="boolean"]
 [type="string"]
 ...
>
```
