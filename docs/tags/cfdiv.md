# cfDiv


Creates an HTML tag with specified contents and lets you to use bind expressions to dynamically control the tag contents.

---
## Body
free

## Example
```
<cfDiv
 [bind="string"]
 [bindonload="boolean"]
 [id="string"]
 [onbinderror="string"]
 [tagname="string"]
> 
 [</cfDiv>]
```
