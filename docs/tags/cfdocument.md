# cfdocument


Creates a PDF document from a text block containing CFML and HTML.

---
## Body
free

```
<cfdocument
 [authpassword="string"]
 [authuser="string"]
 [backgroundvisible="boolean"]
 [bookmark="boolean"]
 [encryption="String"]

 [fontembed="string"]
 [format="string"]
 [htmlbookmark="boolean"]
 [localurl="boolean"]
 [marginbottom="number"]
 [marginleft="number"]
 [marginright="number"]
 [margintop="number"]
 [mimetype="string"]
 [name="string"]
 [orientation="string"]
 [overwrite="boolean"]
 [ownerpassword="string"]
 [pageheight="number"]
 [pagetype="string"]
 [pagewidth="number"]
 [permissions="string"]
 [proxyhost="string"]
 [proxypassword="string"]
 [proxyport="number"]
 [proxyuser="string"]
 [saveasname="string"]
 [scale="number"]
 [src="string"]
 [srcfile="string"]
 [unit="string"]
 [useragent="string"]
 [userpassword="string"]
> 
 [</cfdocument>]
```
