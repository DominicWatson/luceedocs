# cfexecute


Enables developers to execute a process on a server computer.

---
## Body
required

<cfexecute
 [arguments="any"]
 [errorfile="string"]
 [errorvariable="string"]

 [outputfile="string"]
 [terminateontimeout="boolean"]
 [timeout="number"]
 [variable="string"]
> 
 </cfexecute>
