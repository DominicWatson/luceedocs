# cfhttpparam


Required for cfhttp POST operations, cfhttpparam is used to specify the parameters necessary to
	 build a cfhttp POST.

---
## Body
prohibited

## Example
```
<cfhttpparam
 [encoded="boolean"]
 [file="string"]
 [mimetype="string"]
 [name="string"]
 type="body|CGI|cookie|file|form|formfield|header|URL|XML"
 [value="any"]
>
```
