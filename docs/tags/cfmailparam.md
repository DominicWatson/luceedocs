# cfmailparam


Can either attach a file or add a header to a message. It is nested within a cfmail tag. You can
  use more than one cfmailparam tag within a cfmail tag.

---
## Body
prohibited

## Example
```
<cfmailparam
 [content="any"]
 [contentid="string"]
 [disposition="string"]
 [file="string"]
 [name="string"]
 [remove="boolean"]
 [type="string"]
 [value="string"]
>
```
