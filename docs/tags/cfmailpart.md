# cfmailpart


Specifies one part of a multipart e-mail message. Can only be used in the cfmail tag. You can use more than one cfmailpart tag within a cfmail tag

---
## Body
required

## Example
```
<cfmailpart
 [charset="string"]
 type="string"
 [wraptext="number"]
> 
 </cfmailpart>
```
