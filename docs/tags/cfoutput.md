# cfoutput


Displays the results of a database query or other operation. To nest cfoutput tags, see the "Usage" section.

---
## Body
required

## Example
```
<cfoutput
 [endrow="number"]
 [group="string"]
 [groupcasesensitive="boolean"]
 [maxrows="number"]
 [query="object"]
 [startrow="number"]
> 
 </cfoutput>
```
