# cfparam


Tests for a parameter's existence, tests its data type, and provides a default value if one
  is not assigned.

---
## Body
prohibited

## Example
```
<cfparam
 [default="any"]
 [max="number"]
 [maxlength="int"]
 [min="number"]
 name="string"
 [pattern="string"]
 [type="string"]
 ...
>
```
