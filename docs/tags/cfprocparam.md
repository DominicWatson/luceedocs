# cfprocparam


Specifies parameter information, including type, name, value, and length. The cfprocparam tag
  is nested within a cfstoredproc tag.

---
## Body
prohibited

## Example
```
<cfprocparam
 [cfsqltype="string"]
 [dbvarname="string"]
 [maxlength="number"]
 [null="boolean"]
 [scale="number"]
 [sqltype="string"]
 [type="string"]
 [value="any"]
 [variable="string"]
>
```
