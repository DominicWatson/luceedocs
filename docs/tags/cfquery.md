# cfquery


Passes SQL statements to a data source. Not limited to queries.

---
## Body
required

## Example
```
<cfquery
 [blockfactor="number"]
 [cachedafter="datetime"]
 [cachedwithin="object"]
 [cachename="string"]
 [connectstring="string"]
 [datasource="any"]
 [dbname="string"]
 [dbserver="string"]
 [dbtype="string"]
 [debug="boolean"]
 [lazy="boolean"]
 [maxrows="number"]
 [name="string"]
 [ormoptions="struct"]
 [params="object"]
 [password="string"]
 [provider="string"]
 [providerdsn="string"]
 [psq="boolean"]
 [result="string"]
 [returntype="string"]
 [timeout="number"]
 [timezone="string"]
 [unique="boolean"]
 [username="string"]
> 
 </cfquery>
```
