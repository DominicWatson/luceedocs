# cfregistry


Reads, writes, and deletes keys and values in the system registry. The cfregistry tag is supported
  on all platforms, including Linux, Solaris, and HP-UX.

---
## Body
prohibited

## Example
```
<cfregistry
 action="delete|get|getAll|set"
 branch="string"
 [entry="string"]
 [name="string"]
 [sort="string"]
 [type="string"]
 [value="string"]
 [variable="string"]
>
```
