# cftrace


Displays and logs debugging data about the state of an application at the time the cftrace tag executes.

---
## Body
free

## Example
```
<cftrace
 [abort="boolean"]
 [category="string"]
 [follow="boolean"]
 [inline="boolean"]
 [text="string"]
 [type="string"]
 [var="string"]
> 
 [</cftrace>]
```
