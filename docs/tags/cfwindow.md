# cfWindow


Creates a pop-up window in the browser. Does not create a separate browser pop-up instance. 

---
## Body
free

## Example
```
<cfWindow
 [buttons="string"]
 [draggable="boolean"]
 [height="numeric"]
 [initshow="boolean"]
 [jslib="string"]
 [minheight="numeric"]
 [minwidth="numeric"]
 [modal="boolean"]
 [name="string"]
 [onbinderror="string"]
 [refreshonshow="boolean"]
 [resizable="boolean"]
 [source="string"]
 [title="string"]
 [width="numeric"]
 [x="numeric"]
 [y="numeric"]
> 
 [</cfWindow>]
```
